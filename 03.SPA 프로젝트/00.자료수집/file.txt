3. 계획
메인/서브로 구성 
(1) 메인 : 게임요소를 넣기!
메인페이지 주 기능 : 
스크롤 시 캐릭터가 이동하고 캐릭터가 특정위치에 도착하면 그 위치에 대한 설명 및 이미지 출력
버튼을 누르면 다음 단계 진행
진행 단계 : 
캐릭터 도착 -> 이미지 출력 -> 버튼을 눌러서 다음 단계 진행 -> 이미지2 출력 
-> 버튼을 눌러서 해당 섹션 빠져나오기 -> 스크롤해서 다음 위치로 이동
총 5개의 섹션로 구성(출발-챕터1-챕터2-챕터3-챕터4)
=> 상단모듈 + 내용출력모듈 + 내용출력모듈 + 내용출력모듈 + 내용출력모듈 + 하단모듈
내용출력모듈 데이터구조 :
이미지 + 설명 + 진행버튼

(2) 서브1 : 단순 게임 소개 구조
주 기능 : 스크롤 시 게임 소개 출력
모듈화 :
상단모듈 + 요약소개모듈1 + 요약소개모듈2 + 연혁소개모듈 + 링크모듈 + 비디오모듈 + 하단모듈
요약소개모듈1 : 타이틀 + 소개 + 이미지
요약소개모듈2 : 타이틀 + 이미지 + 소개
연혁소개모듈 : 타이틀  + 년도 + 소개
링크모듈 : 링크
비디오모듈 : 비디오 + 설명


(3) 서브2 : 캐릭터 소개
주 기능 : 캐릭터 소개
모듈화 :
상단모듈 + 버튼모듈 + 소개모듈 + 하단모듈
소개모듈 : 이미지 + 소개 + 이전/다음 버튼
